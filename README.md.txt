# Daily Task Reminder (MS Access VBA)

A simple daily task reminder built in Microsoft Access using VBA.

## Features
- Shows a popup of today's incomplete tasks
- Lets you mark tasks complete/incomplete
- Simple form to add and manage tasks

## How to Use
1. Open the `DailyTaskReminder.accdb` in Access.
2. Open the `frmTasks` form.
3. Add or mark tasks as complete.
