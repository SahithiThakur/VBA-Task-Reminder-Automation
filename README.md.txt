# 📅 Daily Task Reminder

A simple MS Access project that helps you track and manage daily tasks.  
It shows a reminder popup for tasks due today and allows marking tasks as complete or incomplete.

---

## 🚀 Features
- 📌 Add, edit, and delete tasks.
- ⏰ Automatic daily popup for incomplete tasks.
- ✅ Mark tasks as complete with a checkbox and submit button.
- 📅 Store and manage all tasks in a central table.
- 🖼️ User-friendly form interface.

## 🛠️ How It Works
1. When you open the database, it automatically shows a popup of today's incomplete tasks.
2. You can add a new task using the **Add New** button.
3. Mark a task as **Done** by ticking the checkbox and clicking **Submit**.
4. Completed tasks are removed from the popup and marked in the table.
